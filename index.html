JavaScript

JavaScript is a programming language used in browsers to create interactive features.

Modulo

A JavaScript operator that returns the remainder of a division problem. Noted with the "%" symbol.

Operators

Arithmetic operators (+, -, *, /, %, etc.) perform their given arithmetic to return a number. There are also comparison operators, which we will not describe here.

NaN

"Not a Number" but treated as a number.

Variable Naming Conventions

For example: lowerCamelcase. JavaScript uses lowerCamelcase, sometimes UpperCamelCase, and promotes using succinct, descriptive variable names.

+=

A shortcut to ammend the value of a variable by the value given after the += operator.

Function

A function is an action or set of actions, not necessarily connected to an object.

Method

A method is an action/actions performed on a specific object.

Argument to a Method

The values that are passed into the method.

String

A string is a set of characters that isn't a number, but can include numbers. For example "This is a string". Strings are 0-indexed (the first character is assigned an index of 0, not 1).

Chaining methods

A "trainwreck". A method returns an object which may in turn have a method called on it. Therefore, methods can be "chained" together.

Escape

A '\' in a string. Tells Javascript that the next character is part of the string, not an operator.

Alert

A function that creates a notification to the user. In chrome this is a pop up window.

Commenting Code

A good coding practice. Comments should be left in the code so you and other programmers can understand the code later.

Booleans

True and False (0 and 1). A primitive variable that is not a string, number, undefined, or NULL.

Undefined

A primitive variable meaning not defined (or nothing was returned from a function).

Parameter

The variable name assigned to a given arguement.

Return

Produces a primitive variable from a function.
